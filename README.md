# Student Performance Analysis Dashboard

## Project Overview

This project analyzes student academic performance using Excel and Power BI.

The dashboard helps identify factors affecting student scores and provides insights into academic performance trends.

---

## Objectives

- Analyze student scores
- Measure impact of test preparation
- Study parent education influence
- Classify students by performance category
- Build interactive dashboard

---

## Tools Used

- Microsoft Excel
- Power BI
- GitHub

---

## Key Metrics

- Average Student Score
- Highest Score
- Lowest Score
- Total Students

---

## Dashboard Features

- Gender-wise Performance Analysis
- Test Preparation Impact Analysis
- Parent Education Analysis
- Performance Distribution
- Interactive Filters

---

## Key Insights

- Students who completed test preparation scored higher.
- Parent education level showed positive impact on performance.
- Most students belong to the Good performance category.
- Female students performed better in reading and writing scores.
